# CampusLibraryBorrowing-tracker
 a simple swiftUI library borrowing management app using Coore data.  you can add books add members , borrow books , etc ....

SETUP : 

first open campuslibrary.xcodeproj in Xcode
then make sure the target is set to your Mac or simulator
Build and run the project

the app uses Core Data;
the initial seed data includes : 
category: fiction
book : "1984" by George Orwell 
i added too another book called rich dad poor dad by robert kiosaky just to test it
